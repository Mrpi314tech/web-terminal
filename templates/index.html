Basic commands:

pwd - displays the current working directory
cd - change the current working directory
ls - list the files and directories in the current working directory
mkdir - create a new directory
touch - create a new file
cp - copy a file or directory
mv - move a file or directory
rm - delete a file or directory
chmod - change the permissions of a file or directory
echo - display a message on the terminal
cat - display the contents of a file
grep - search for a specific pattern in a file or multiple files
find - search for files and directories based on certain criteria
wc - display the number of lines, words, and characters in a file
